# NEO_BLA_BLA - Plataforma Web de Idiomas

## ▶️ ¿Cómo ejecutar?

### 1. Frontend
```bash
cd frontend
npm install
npm start
```

### 2. Backend
```bash
cd backend
npm install express cors node-fetch dotenv
node server.js
```
